# Comprehensive Building Systems Documentation

## Building Types
1. **Residential Buildings**  
   - Description of residential buildings.
   - Examples: Houses, Apartments.

2. **Commercial Buildings**  
   - Description of commercial buildings.
   - Examples: Stores, Offices.

3. **Industrial Buildings**  
   - Description of industrial buildings.
   - Examples: Factories, Warehouses.

## Progression Stages
- **Stage 1:** Basic construction capabilities.  
- **Stage 2:** Intermediate building techniques.  
- **Stage 3:** Advanced construction methods and styles.

## Resource Costs
- **Wood:** Required for all building types.  
- **Stone:** Required for sturdier structures.  
- **Metal:** Needed for industrial and futuristic designs.

## Mechanics
- **Resource Gathering:** Players must gather resources to build.
- **Construction Timers:** Different building types have different construction times.
- **Upgrades:** Buildings can be upgraded to enhance functionality and aesthetics.
